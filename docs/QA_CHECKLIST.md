# QA CHECKLIST

## Desktop checks

- [ ] オンボーディングが初回のみ表示される
- [ ] 「カメラを許可する」導線が見やすい
- [ ] カメラ許可で表情認識が開始される
- [ ] カメラ拒否時に日本語の復旧案内が出る
- [ ] 「もう一度カメラを確認」で再試行できる
- [ ] 「あそび方」から練習画面を開ける
- [ ] 練習画面で 1 表情ずつ確認できる
- [ ] タイトルの各ボタンが押しやすく動作する
- [ ] 着せ替え画面のキャラ / 背景切り替えが動作する
- [ ] ガチャ画面の表示と結果メッセージが自然な日本語になっている
- [ ] プレイ中のタップ操作ボタンが使える
- [ ] 表情認識が使えないときに非ブロッキング案内が出る
- [ ] 結果画面で 共有 / もう一度 / タイトルへ が動作する
- [ ] プライバシー画面の文言が実装と一致している
- [ ] 設定とデータ画面からチュートリアルリセットが動作する
- [ ] 設定とデータ画面からデータリセット確認が動作する

## iPhone checks

- [ ] 初回起動時のオンボーディングが短く分かりやすい
- [ ] カメラ許可ダイアログの前に日本語説明が出る
- [ ] カメラ拒否後に復旧案内が全画面で出る
- [ ] 実機で笑顔 / 怒った顔 / 驚いた顔 / 悲しい顔が認識される
- [ ] 表情認識が不安定でもタップ操作で最後まで遊べる
- [ ] 縦画面でノッチやホームインジケータに UI が重ならない
- [ ] HUD の文字が小さすぎない
- [ ] タッチボタンが押しやすい
- [ ] 結果画面のボタンが押しやすい
- [ ] 共有フローが動作する
- [ ] アプリを再起動しても保存データが維持される
- [ ] チュートリアルリセット後、次回開始時に練習が再表示される
- [ ] データリセット後、初回状態に戻る
- [ ] 機内モードまたは不安定な通信環境でもアプリが使える
- [ ] App Store 用スクリーンショットとして見栄えがする
